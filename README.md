
## Concepts Used

- **React.memo** – Prevents unnecessary component re-renders.
- **useMemo** – Memoizes expensive calculations (product filtering).
- **useCallback** – Memoizes functions to prevent child re-renders.
- **Redux Toolkit**
  - `configureStore`
  - `createSlice`
  - `createAsyncThunk`

## Tech Stack

- React
- Redux Toolkit
- React Redux
- JavaScript (ES6)

## Learning Goal

This project was created to understand:

- React rendering optimization
- Memoization in React
- Redux async flows using `createAsyncThunk`
- Connecting API calls with Redux state
